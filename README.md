# 💰 Finance Dashboard

A responsive finance dashboard built using React, Vite, and Tailwind CSS.

## 🚀 Features
- 📊 Dashboard summary (Balance, Income, Expenses)
- 📈 Charts (Line chart & Pie chart)
- 📋 Transactions table (Search, Filter, Sort)
- 🔐 Role-based UI (Admin / Viewer)
- 🌙 Dark Mode support
- 💾 Local Storage persistence

## 🛠️ Tech Stack
- React
- Vite
- Tailwind CSS
- Recharts

## 📱 Responsiveness
- Fully responsive (Mobile + Tablet + Desktop)
